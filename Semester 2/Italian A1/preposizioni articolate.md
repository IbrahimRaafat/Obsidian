Below you can see how we form the "preposizioni articolate” for the prepositions “da” and “a".

|         |        |       |
| ------- | ------ | ----- |
| **+**   | **DA** | **A** |
| **IL**  | dal    | al    |
| **LO**  | dallo  | allo  |
| **LA**  | dalla  | alla  |
| **L'**  | dall'  | all'  |
| **I**   | dai    | ai    |
| **GLI** | dagli  | agli  |
| **LE**  | dalle  | alle  |
